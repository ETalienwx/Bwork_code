接口 方法 作用 备注（参数） 返回值
/ get  查看拨测api文档    
/health get  查看服务是否alive   { "version": "1.0.0", "status": "alive" }
/createtask get  显示创建任务界面    
/createtask post  创建任务 采用form数据格式  { "ip": "183.146.25.70", "port": "1935" } { "id": "srt", "name": "str", "type": "srt", "cycle": "srt", "detail": { "ip_ports": [ "183.146.25.70:1935" ] } }
/createtasks get  显示创建任务页面    
/createtasks post  创建任务  ["127.0.0.1:1935","127.0.0.1:1935"]  
/tasks get  获取所有的task     { "id": "5f1a672c986f8f1926e4f170", "creat_time": "2020-07-24 11:45:39" }
/getresult get  显示查询任务结果界面    
/getresult post  查询任务结果 采用form数据格式  { "task_id": "5f1a4f351d8a7b923bf687f0" } { "required_bitrate": 4.09, "description": "高清 1080P+", "ping_ts": 1594230498, "download_speed": 135.97, "download_ts": 1594230498, "city": null, "task_id": "5f06069cccf0ed0f6970cdc0", "id": "5f0606e5ccf0ed0f6970cdc1" }
/getresult/{task_id} get  获取当前task_id的任务结果    
/getstatus get  显示查询状态页面    
/getstatus post  查询任务状态 采用form数据格式  { "task_id": "5f1a4f351d8a7b923bf687f0" } { "task_id": "5f1a4f351d8a7b923bf687f0", "status": "finish" }
/getstatus/{task_id} get  获取当前task_id的任务状态    
/stop get  显示停止任务界面    
/stop post  停止任务 采用form数据格式  { "task_id": "5f1a4f351d8a7b923bf687f0" }
